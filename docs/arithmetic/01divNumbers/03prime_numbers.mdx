Число $1$ имеет только один делитель - единицу. Любое другое натуральное число имеет, по крайней мере, два делителя - это само число и $1$. <br />
Например, числа $2,3,5,7,11,13, ...$ имеют только по $2$ делителя: единицу и самого себя.

<TeX math="3:3=1;\quad 3:1=3." block />

У числа $12$ делителей больше двух - это числа $1,2,3,4,6,12.$
:::info Определение
Натуральное число называют **простым**, если оно имеет **только два натуральных делителя**: единицу и само это число.<br />
Натуральное число называют **составным**, если оно имеет **больше двух натуральных делителей**.
:::
